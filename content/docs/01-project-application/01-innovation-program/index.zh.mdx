# 大学生创新创业训练计划项目

大学生创新创业训练计划项目（简称"大创"）旨在加强学生的创新技能和创业培训，提高他们适应教育改革和国家发展对高水平人才需求的能力。

## 项目时间线和结构

该计划通常遵循年度周期：
- **申请期**：通常每年5月份开始
- **项目周期**：自批准后为期两年
- **实施阶段**：
  1. **项目申报与立项**
  2. **中期检查**
  3. **结题报告**

## 申请流程

1. **学校通知**：学校发布关于申请期的通知
2. **初步准备**：学生准备基本项目提案并寻找学院内的指导教师
3. **系统提交**：学生通过在线平台提交基本项目信息和指导教师详情
4. **教师审批**：指导教师登录系统审核并批准申请
5. **学院评审**：学院统一对所有申请进行评审
6. **项目分类**：根据分配名额，项目被分类为校级、省级或国家级

## 经费结构

项目根据其分类级别获得不同的经费：
- **校级**：5,000元
- **省级**：10,000元
- **国家级**：20,000元

## 经费发放

经费分三个阶段发放：
- **初始阶段**：总经费的30%（项目批准后即可使用）
- **中期阶段**：总经费的50%（通过中期检查后发放）
- **完成阶段**：总经费的20%（项目成功结题后发放）

## 项目名额分配

各学院获得的项目名额基于：
- 学生人数
- 荣誉积分/过往表现
- 其他机构因素
