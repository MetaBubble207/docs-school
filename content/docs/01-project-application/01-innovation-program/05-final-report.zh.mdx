# 🎯 结题验收流程

结题验收是大学生创新创业训练计划项目的最后一个重要阶段。它标志着项目的正式完成，并对项目成果、取得的成就和发现进行全面总结与评估。

## 📅 时间安排

> 📢 **重要提示**：每年的具体时间节点可能有所变化，请以教务部网站发布的最新通知为准：[教务部实践通知](https://jwb.dgut.edu.cn/tzgg/sjtz.htm)

参考近年的结题验收时间安排：
- 📝 **系统开放**：12月初（如2024年为12月2日）
- ⏰ **学生提交截止**：次年2月中旬（如2025年为2月16日）
- 📋 **指导教师审核截止**：次年2月下旬（如2025年为2月23日）
- 🏫 **学院审核截止**：次年3月上旬（如2025年为3月3日）
- 🎤 **答辩评审**：次年3月中下旬（具体时间另行通知）
- 💰 **经费报销截止**：次年5月（如2025年5月）

## 💼 结题验收的重要性

结题验收有几个关键作用：

1. **完整性验证**：确认项目按照计划完成并达成预期目标
2. **成果评估**：评价项目成果的质量、创新性和影响力
3. **知识分享**：为后续研究和相关项目提供参考和借鉴
4. **资金使用审核**：核实项目资金的合理使用情况
5. **能力培养检验**：评估学生在项目中的成长和能力提升

> ⚠️ **特别提醒**：必须通过结题验收才能完成最终剩余经费的报销！项目负责人需在规定时间内（如2025年5月前）完成经费核销。

## 📋 系统操作指南

### 步骤1：登录系统

1. 🌐 访问大创系统：[http://180.108.46.32:81/dglgxy_cx/](http://180.108.46.32:81/dglgxy_cx/)
2. 🔑 使用学号和密码登录
3. 📋 在左侧菜单找到"提交结题表"选项

### 步骤2：填写结题报告

根据系统界面，需要填写以下内容：

1. 📝 **项目基本信息**
   - 项目名称
   - 项目类型（创新训练/创业训练/创业实践）
   - 项目级别（校级/省级/国家级）
   - 项目负责人
   - 指导教师

2. 📊 **项目完成情况**
   - 项目研究目标达成情况
   - 项目研究内容完成情况
   - 项目研究过程及方法
   - 取得的主要研究成果

3. 💰 **经费使用情况**
   - 已使用经费金额
   - 经费使用明细
   - 经费使用合理性说明

4. 🏆 **研究成果清单**
   - 发表的论文/专利
   - 参加的比赛/获得的奖项
   - 制作的实物成果/软件作品
   - 其他形式的研究成果

5. 📚 **项目总结与反思**
   - 项目创新点
   - 存在的不足
   - 经验与感悟
   - 未来展望

### 步骤3：上传研究成果

除填写结题报告外，还需上传项目成果和支持材料：

1. 📄 **结题报告完整版**（PDF格式）
   - 系统会自动生成报告，确认无误后下载
   - 建议自行备份一份

2. 📑 **研究成果证明材料**
   > ⚠️ **重要提示**：系统只允许上传**一个**附件文件，如有多个材料需合并为一个ZIP压缩包

   压缩包中可包含以下材料：
   - 论文全文或录用通知
   - 专利申请受理通知书或授权证书
   - 获奖证书扫描件
   - 实物成果照片或视频（控制文件大小）
   - 软件著作权证书

3. 💻 **软件/代码提交说明**
   - 如项目成果为软件，可以只提交核心代码文件
   - 代码应包含必要注释，便于评审专家理解
   - 可附带简短的安装/使用说明文档
   - 如代码量较大，建议只提交关键部分并附README说明
   - 确保压缩文件大小符合系统限制（通常不超过20MB）

4. 📊 **文件组织建议**
   - 为压缩包中的文件使用清晰的命名
   - 按类别创建不同文件夹（如"论文"、"专利"、"获奖证书"等）
   - 添加一个索引文档，列出所有文件及其简要说明
   - 确保重要文件放在明显位置

> 💡 **技巧**：在提交前，请确保压缩包可以正常打开，所有文件完好无损。最好提前让其他人测试下载和打开，确保评审专家能够顺利查看您的材料。

### 步骤4：提交与审核流程

1. 📧 **学生提交**
   - 在截止日期前完成系统提交
   - 确保所有材料完整准确
   - 提交后及时通知指导教师审核

2. 👨‍🏫 **指导教师审核**
   - 指导教师登录系统审核
   - 填写审核意见
   - 在截止日期前完成审核
   - 审核内容包括：
     - 项目完成质量评价
     - 成果水平评价
     - 团队表现评价

3. 🏫 **学院初审**
   - 学院组织专家评审
   - 在截止日期前完成审核
   - 提交教务部备案
   - 初审内容包括：
     - 项目完成情况综合评价
     - 研究成果真实性与价值评价
     - 项目创新性评价

4. 🎓 **校级验收评审**
   - 国家级和省级项目参加学校组织的专家评审（答辩形式）
   - 校级项目一般由学院负责最终评审

## 📑 结题材料准备要求

### 1. 结题报告书

准备一份全面的结题报告（一般15-20页），包括：

- **项目概述**：项目背景、目标、意义的简要总结
- **研究方法**：详细说明采用的研究方法和技术路线
- **研究过程**：描述研究的具体过程和步骤
- **结果与分析**：展示研究结果并进行分析
- **创新点**：阐述项目的创新之处和特色
- **问题与解决方案**：研究过程中遇到的困难及解决办法
- **结论**：总结研究的主要发现和结论
- **参考文献**：列出研究中引用的文献资料

### 2. 研究成果材料

根据项目特点，准备相应的研究成果材料：

- **论文类**：已发表或录用的学术论文全文、录用通知
- **专利类**：专利申请受理通知书、专利授权证书
- **作品类**：软件著作权证书、设计作品照片、使用说明
- **获奖类**：比赛获奖证书、表彰文件
- **应用类**：产品应用证明、用户反馈、效益分析

### 3. 经费使用记录

提供详细的经费使用记录：

- **经费预算对照表**：对比原计划预算和实际支出
- **支出明细清单**：列出所有项目支出，包括用途说明
- **发票和收据**：保存所有支出的原始凭证

## 🔍 评审标准与评级

### 评审标准

结题验收评审主要基于以下几个方面：

1. **目标达成度** (30%)
   - 预期目标的完成情况
   - 研究内容的完整性
   - 技术路线的合理性

2. **成果质量** (30%)
   - 研究成果的学术/应用价值
   - 成果的创新性
   - 成果的完成度

3. **研究过程** (20%)
   - 研究方法的科学性
   - 过程管理的规范性
   - 问题解决的能力

4. **团队表现** (10%)
   - 团队合作情况
   - 成员能力提升
   - 指导教师参与度

5. **经费使用** (10%)
   - 经费使用的合理性
   - 财务记录的规范性
   - 资源利用效率

### 评级标准

根据历年结题验收情况，项目评级通常分为：

1. **优秀**：
   - 完全达成预期目标，甚至超出预期
   - 成果具有显著创新性和应用价值
   - 形成高质量的论文、专利或实物成果
   - 获得省级以上比赛奖项

2. **良好**：
   - 基本达成预期目标
   - 成果具有一定创新性
   - 形成质量较好的研究成果
   - 可能获得校级或市级比赛奖项

3. **合格**：
   - 达成大部分预期目标
   - 成果完整但创新性一般
   - 形成基本的研究成果
   - 无明显不足之处

4. **不合格**：
   - 未达成主要预期目标
   - 成果存在明显缺陷
   - 研究过程不规范
   - 存在严重问题导致结题失败

> 📊 **参考数据**：根据历年结题评审情况，如2014年度的结题验收中，共有50个省级以上项目参加结题验收答辩，其中9个项目获得优秀、15个项目获得良好、26个项目获得合格通过。

## 🎤 答辩准备指南

### 答辩对象

国家级和省级项目需要参加学校组织的结题答辩。校级项目一般由学院负责组织结题评审。

### 答辩准备材料

1. 📊 **PPT演示文稿**（10-15分钟）
   - 项目概述（背景、目标、意义）
   - 研究方法与过程
   - 主要研究成果
   - 创新点与贡献
   - 团队分工与合作

2. 🧩 **实物演示**（如适用）
   - 产品原型
   - 系统演示
   - 软件展示

3. 📝 **补充材料**
   - 研究数据
   - 详细分析结果
   - 相关证书或证明

### 答辩流程

1. 🎯 **项目陈述**：10-15分钟，介绍项目情况
2. ❓ **专家提问**：5-10分钟，回答评审专家的问题
3. 💬 **评审讨论**：专家组内部讨论评分
4. 📝 **结果反馈**：答辩结束后统一公布结果

## 💡 成功结题的技巧

1. 📚 **提前规划**：制定合理的结题时间表，留出充足准备时间
2. 📑 **资料完整**：确保所有必要文档和材料齐全
3. 🔍 **严谨核对**：仔细检查报告内容，避免错误和遗漏
4. 📊 **突出亮点**：强调项目的创新点和主要成果
5. 💡 **诚实总结**：客观分析问题和不足，展示解决方案
6. 👨‍🏫 **教师指导**：充分征求指导教师的意见和建议
7. 🔄 **反复修改**：多次修改完善结题报告，确保质量
8. 🎯 **答辩演练**：提前准备并演练答辩过程

## ⚠️ 特殊情况处理

### 项目延期

如因特殊原因无法按时完成项目，可申请延期结题：

1. 📝 **申请流程**：
   - 下载并填写《项目延期申请书》（教务部网站可下载）
   - 经指导教师签字后提交学院审核
   - 学院审核通过后报教务部审批

2. ⏱️ **延期要求**：
   - 延期时间最多不超过半年
   - 必须有正当理由（如实验设备故障、材料延迟到货等）
   - 一个项目原则上只能申请一次延期

### 项目终止

如确实无法完成项目，可申请终止：

1. 📄 **申请材料**：
   - 项目终止申请书
   - 已完成工作说明
   - 经费使用情况说明

2. 💼 **注意事项**：
   - 终止项目不再安排后续经费
   - 已使用经费需提供合理说明
   - 可能影响团队成员今后申请资格
