# 大学生创新创业训练计划项目

为进一步强化大学生创新创业能力培训，增强学生的创新意识和创业能力，培养适应教育改革和国家发展需要的高水平人才，学校根据省教育厅《关于开展2020年度国家级省级大学生创新创业训练计划立项工作的通知》，组织校内大学生创新创业训练计划项目申报工作。

## 项目类别

大学生创新创业训练计划项目分为创新训练项目、创业训练项目和创业实践项目三类：

1. **创新训练项目** - 本校在校生个人或团队，在导师指导下，完成创新性研究项目设计、研究准备、项目实施、研究报告撰写、成果公开发表等工作。

2. **创业训练项目** - 本校在校生团队，在导师指导下，团队中每位学生在项目实施过程中承担一个特定的角色，完成编制商业计划书、开展可行性研究、模拟企业运行、企业实践、撰写创业报告等工作。

3. **创业实践项目** - 本校学生团队，在校内、企业导师共同指导下，在前期创新训练项目（或实验）成功的基础上，向前迈出市场应用前景好的创新性产品或服务方案，以此为基础开展创业实践活动。

## 经费支持

校级项目经费资助标准为5000元/项。经费可用于项目研究所需的材料费、实验测试费、会议费、资料费、办公文具费及专家咨询费、成果推广确定的必要性支出、项目组成员补助、差旅费等费用。

## 项目申报要求

1. **项目团队**：申报人必须是在校学生，每个项目团队人数不超过5人，鼓励跨专业、跨学科等团队开展项目。

2. **计划项目执行时间**：一般为1年。

3. **指导教师**：创新训练计划项目的校内指导教师应具有中级以上职称或硕士及以上学位；创业训练和创业实践项目的校内第一指导教师应具有中级以上职称。

4. **申报流程**：在计划项目申报系统（http://180.108.46.32:81/dglgxy_cx/）完成。学生可以通过账号密码登录方式或一键登录直接进入系统。
