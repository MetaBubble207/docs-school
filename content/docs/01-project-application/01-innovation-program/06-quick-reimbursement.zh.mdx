# 快速报销指南

本指南为创新创业训练计划项目提供了一套简明的报销处理方法。请按照以下步骤确保您的报销申请能够快速、准确地处理。

## 报销流程一览

| 步骤 | 操作         | 详细说明                                  |
| ---- | ------------ | ----------------------------------------- |
| 1    | 购买东西     | 单价最好 300 以下，300 以上要列为固定资产 |
| 2    | 获取有效发票 | 获取印有学校名称的正规发票                |
| 3    | 系统填报业务 | 注意要发票查验                            |
| 4    | 整理报销材料 | 按支出类别整理收据                        |
| 5    | 提交材料     | 提交至松山湖行政楼 1A208                  |

## 购买前的准备工作

1. **确认符合条件**：确认计划的支出属于允许的类别
   目前已经确定可以购买的物品清单有：显示器、鼠标、键盘、耳机、排插、硬盘、内存条、单片机、
2. **检查预算可用性**：确保项目预算中有足够的资金
   特别是多项目计算的时候，购买物资得按照单独项目计算，不能按照能够报销多少个项目的总金额去购买东西
3. **记录购买理由**：说明购买与项目目标的关联性

## 发票要求

所有发票必须：

- 是正规税务发票
- 注明东莞理工学院作为购买方
- 清楚标明购买内容
- 包含日期、金额和供应商信息
- 在项目时间范围内开具
- 是原件（非复印件或照片）

## 报销步骤详解

### 1. 收集并整理材料

- 按支出类别（材料、测试、差旅等）分组收据
- 附上支持材料（价格比较、说明文件等）
- 为所有文件制作数字副本作为记录

### 2. 填写报销表格

- 通过财务平台获取表格
- 按照审批文件中的确切信息输入项目详情
- 使用正确的类别代码列出每项支出
- 仔细核对所有金额和描述
- 获取必要的签名（项目负责人、指导教师）

### 3. 通过财务平台提交

1. 登录一站式服务平台：[https://ywtb.dgut.edu.cn/custom/index.html#/](https://ywtb.dgut.edu.cn/custom/index.html#/)
2. 导航至"系统直达"并选择"财务平台"
3. 选择"报销申请"→"学生项目报销"
4. 上传所有必需文件：
   - 填写完整的报销表格
   - 扫描的原始发票
   - 支持文件
5. 提交申请并记录跟踪号

### 4. 监控和跟进

- 通过财务平台定期检查状态
- 及时回应任何要求提供额外信息的请求
- 一旦批准，资金通常在 7-10 个工作日内转账

## 常见问题及快速解决方案

### 问题：发票上没有学校名称

**解决方案**：返回供应商处要求重新开具正确购买方信息的发票

### 问题：需要紧急购买但没有资金

**解决方案**：团队成员可以使用个人资金购买，之后再报销（确保获得正规发票）

### 问题：不确定使用哪个预算类别

**解决方案**：查阅详细的经费指南或联系指导教师获取指导

### 问题：财务办公室退回了您的申请

**解决方案**：仔细阅读他们的意见，按要求进行修改，并及时重新提交

## 高效报销的最佳实践

1. **批量提交**：尽可能将多项支出合并为一次提交
2. **保持清晰记录**：维护跟踪所有支出和报销状态的电子表格
3. **定期跟进**：每周检查申请状态
4. **提前规划**：留出至少 2-3 周的报销处理时间
5. **团队沟通**：确保所有团队成员了解正确的流程
