# 🚀 项目申报流程

本指南提供了关于如何申请大学生创新创业训练计划项目的详细信息。不用担心，虽然看起来步骤很多，但按照这个攻略一步步来，你也能成功通关！

## 📅 申请时间表

> 📢 **重要提示**：每年的具体时间节点可能有所变化，请以教务部网站发布的最新通知为准：[教务部实践通知](https://jwb.dgut.edu.cn/tzgg/sjtz.htm)

参考近年的申报时间安排：
- 📝 **申报通知发布**：4月-5月（如2025年为4月10日）
- ⏰ **项目申报期**：通知发布后约1个月（如2025年截止日期为5月16日）
- 📋 **立项公示期**：5-7天
- 🎉 **最终结果公布**：通常在公示期结束后1-2周内

## 🎯 申请准备工作

在冲击大创之前，先做好这些准备：

1. 💡 **头脑风暴项目点子**
   - 找一个既有创新性又能落地的点子（太简单没意思，太难实现不可行）
   - 确保项目与你的专业或兴趣相关（这样才不会半路想放弃）
   - 考虑社会价值和创新点（评审老师最爱看这个）

2. 👥 **组建梦之队**
   - 招募有能力且靠谱的队友（不是只找朋友，要找能干活的人！）
   - 团队最多5名学生（多了可能会变成"五龙争霸"）
   - 最好有不同专业背景的成员（跨学科合作很加分）

3. 📝 **准备申请材料**
   - 项目详细计划（不是三言两语就能糊弄过去的）
   - 创新点和技术路线（这个很重要，评审时的重点关注）
   - 项目可行性分析（证明你不是在做白日梦）
   - 预期成果（别太夸张，但也别太保守）

## 📋 系统申报内容详解

### 1️⃣ 基本信息填写

- 📌 **项目名称**：简洁明了，突出创新点
- 🏫 **所属学院**：选择项目负责人所在学院
- 📚 **学科一级门**：选择项目所属学科大类
- 📖 **学科二级类**：选择具体学科方向
- ⏳ **项目期限**：2025-04-11 至 2027-04-30（固定期限）

### 2️⃣ 团队成员信息

- 👥 **项目成员**：
  - 主持人：填写学号
  - 成员分工：明确每个成员的具体职责
  - 成员类型：选择相应角色

- 👨‍🏫 **指导教师**：
  - 通过输入教师姓名或工号搜索
  - 点击"搜索"按钮选择教师

### 3️⃣ 项目内容填写

1. 📝 **项目简介**（200字以内）
   - 研究目的
   - 主要研究内容

2. 📊 **负责人科研经历**
   - 曾经参与科研的情况

3. 👨‍🏫 **指导教师信息**
   - 承担科研课题情况
   - 对本项目的支持情况

4. 📋 **详细研究计划**
   - 研究目的
   - 研究内容
   - 国、内外研究现状和发展动态
   - 创新点与项目特色
   - 技术路线、拟解决的问题及预期成果
   - 项目研究进度安排

5. 🏗️ **研究基础**
   - 与本项目有关的研究积累和已取得的成绩
   - 已具备的条件
   - 尚缺少的条件及解决方法

## 🛣️ 正确的申请路径

### 步骤1：系统提交申请

1. 🌐 访问项目系统：[http://180.108.46.32:81/dglgxy_cx/](http://180.108.46.32:81/dglgxy_cx/)
2. 🔑 使用学号和密码登录
3. 📋 在左侧仪表盘上找到并点击"项目申报"按钮
4. 📝 在新的页面选择项目类型（创新训练/创业训练/创业实践）
5. ✍️ 按照上述要求填写所有信息

<img src="/images/application/项目申报页面.png" alt="项目申报页面"/>

### 步骤2：寻找指导教师

1. 👨‍🏫 选择合适的指导教师（这步真的超级重要！）
   - 找与项目方向相关的老师
   - 尽量找热心支持学生的老师
   - 资格要求：
     - 创新训练项目：至少具有中级以上职称或博士学位
     - 创业训练和实践项目：至少具有中级以上职称
2. 📝 准备一份简洁的项目介绍
3. 📅 预约面谈时间，向老师展示你的项目想法
4. 👍 获取老师的同意和建议

### 步骤3：指导教师系统确认

1. 📧 提交申请后，系统会通知你的指导教师
2. 📞 主动联系老师，请他们登录系统进行审核
3. ✏️ 老师可能会要求你修改申请内容
4. ✅ 老师确认后，你的申请才会进入学院评审环节
5. ⚠️ 没有指导教师的确认，申请将无法进入下一阶段（卡关了）

### 步骤4：学院评审与立项

1. 👥 学院会组织专家评审所有申请项目
2. 📊 根据创新性、可行性和学术价值进行评分
3. 🏆 按分数和名额分配确定立项级别：
   - 校级项目：相对容易获批，结题要求较低
   - 省级项目：竞争较大，要求较高，需答辩
   - 国家级项目：最高荣誉，竞争最激烈，要求最严格

## 🎉 立项后的行动

恭喜过关！立项后你需要：

1. 🚀 **项目启动**
   - 组织团队会议，细化任务分工
   - 制定详细的工作计划
   - 准备初始材料和设备采购清单

2. 💰 **资金到账**
   - 初始资金（总额的30%）将会发放
   - 按照预算计划合理使用经费
   - 保存所有发票和购买凭证（超级重要！）

3. 📈 **开始实施**
   - 正式开始你的项目研究/开发
   - 定期记录项目进展
   - 与指导老师保持沟通

## 💡 提高申请成功率的秘诀

1. 📚 **做足功课**：了解往年成功项目的特点（可以去找学长学姐取经）
2. 💡 **突出创新**：评审中最重要的就是创新性，没有之一！
3. ⚖️ **保持现实**：提出可行的目标，不要过于雄心勃勃
4. ✍️ **精心设计**：申请书要逻辑清晰，专业用语准确
5. 👨‍🏫 **找对导师**：一个支持你并有相关专业背景的导师是成功一半
6. ⏰ **提早行动**：越早准备，成功率越高（临时抱佛脚真的不行）
7. 🔍 **注重细节**：拼写错误、格式混乱等小问题可能会降低你的专业形象

## ⚠️ 为什么申请会被拒？

避开这些"雷区"：
1. 💤 **创新不足**：太普通、太老套的项目容易被淘汰
2. 🌌 **不切实际**：目标太宏大，两年内明显完不成
3. 🔀 **表达混乱**：申请书逻辑不清，让评审摸不着头脑
4. 💸 **预算不合理**：经费分配不符合常识或规定
5. 👥 **团队不匹配**：成员背景与项目需求不符
6. 👨‍🏫 **指导教师不合适**：老师专业方向与项目关系不大
7. ❌ **申请不完整**：材料缺失或填写不全

记住，大创项目虽然申请过程有点繁琐，但一旦获批，不仅有经费支持你的想法，还能为你的简历增添重量级的一笔。所以，加油吧，未来的创新创业达人！ 🎓✨
