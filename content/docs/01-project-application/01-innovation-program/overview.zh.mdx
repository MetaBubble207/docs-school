# 大学生创新创业训练计划项目概述

大学生创新创业训练计划项目旨在加强学生的创新技能和创业培训，提高他们适应教育改革和国家发展对高水平人才需求的能力。

## 项目类别

大学生创新创业训练计划项目分为三类：

1. **创新训练项目** - 本校在校生个人或团队，在导师指导下，完成创新性研究项目设计、研究准备、项目实施、研究报告撰写、成果公开发表等工作。

2. **创业训练项目** - 本校在校生团队，在导师指导下，团队中每位学生在项目实施过程中承担一个特定的角色，完成编制商业计划书、开展可行性研究、模拟企业运行、企业实践、撰写创业报告等工作。

3. **创业实践项目** - 本校学生团队，在校内、企业导师共同指导下，在前期创新训练项目（或实验）成功的基础上，向前迈出市场应用前景好的创新性产品或服务方案，以此为基础开展创业实践活动。

## 项目规模

每年，该计划旨在实施约360个校级项目。在这些项目中，约30个将被选为国家级项目，约90个将被选为省级项目。

## 经费支持

校级项目经费资助标准为5000元/项。经费可用于：

- 材料费
- 实验测试费
- 会议费
- 资料费
- 办公文具费
- 专家咨询费
- 成果推广费
- 项目组成员补助
- 差旅费

## 知识产权

项目成果中专门技术成果和专门知识的知识产权归学校所有。项目经费由学校提供给学生作为第一作者或发明人。

## 项目益处

参与大学生创新创业训练计划项目提供了众多好处：

1. **技能发展** - 增强研究、创新和创业技能
2. **实践经验** - 获得项目管理的实践经验
3. **学术认可** - 有机会发表研究成果
4. **职业发展** - 培养雇主重视的技能
5. **人脉拓展** - 与教师和行业专业人士建立关系
