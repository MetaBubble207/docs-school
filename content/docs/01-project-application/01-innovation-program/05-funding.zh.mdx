# 经费分配和报销

大学生创新创业训练计划项目为获批项目提供财务支持。了解经费如何分配、哪些支出符合条件以及如何操作报销流程对于有效的项目管理至关重要。

## 经费结构

项目根据其分类级别获得不同的经费金额：

| 项目级别 | 经费金额 |
|---------|---------|
| 校级 | 5,000元 |
| 省级 | 10,000元 |
| 国家级 | 20,000元 |

## 经费发放时间表

经费分三个阶段发放，与项目关键里程碑保持一致：

1. **初始阶段**：总经费的30%
   - 项目批准后立即可用
   - 用于启动成本和初始材料

2. **中期阶段**：总经费的50%
   - 通过中期检查后发放
   - 支持项目的主要实施阶段

3. **完成阶段**：总经费的20%
   - 项目成功结题并最终报告获批后发放
   - 用于最终费用和文档成本

## 符合条件的支出

项目经费可用于以下类别的支出：

### 1. 研究材料和设备
- 实验室用品和消耗品
- 小型设备（每件800元以下）
- 软件许可证（项目实施所需）
- 图书和参考资料（与项目直接相关）

### 2. 测试和实验成本
- 原型开发材料
- 实验室测试费用
- 设备使用费
- 数据收集成本

### 3. 会议和出版费用
- 会议注册费（用于展示项目成果）
- 海报打印费用
- 学术论文出版费
- 知识产权注册费用

### 4. 文档和办公用品
- 项目报告的打印和复印费用
- 专业文具和办公用品
- 装订和展示材料

### 5. 专家咨询费
- 专业咨询服务
- 行业专家访谈
- 技术审查服务

### 6. 团队成员补贴
- 对团队成员工作的有限补偿
- 通常上限为项目总经费的20%
- 必须根据记录的贡献分配

### 7. 差旅费
- 实地研究旅行费用
- 项目相关会议的旅行
- 相关设施的实地考察

## 不符合条件的支出

以下支出通常不符合报销条件：

- 个人电脑或平板电脑
- 手机或通信设备
- 与项目无关的通用软件
- 家具或办公室装饰
- 娱乐费用
- 礼品或捐赠
- 常规学费或课程费用
- 项目批准前产生的费用
- 与项目不直接相关的项目
- 将成为个人财产的物品

## 报销流程

### 步骤1：访问财务平台

1. 访问一网通办平台：[https://ywtb.dgut.edu.cn/custom/index.html#/](https://ywtb.dgut.edu.cn/custom/index.html#/)
2. 点击"系统直通车"
3. 选择"财务平台"
4. 使用你的大学凭证登录

### 步骤2：准备所需文档

在提交报销申请之前，收集以下材料：

1. **原始收据（发票）**
   - 必须是官方的、经税务验证的发票
   - 必须包括大学名称作为购买实体
   - 必须在项目期限内
   - 应清楚地标明购买的内容

2. **报销表格**
   - 完成标准大学报销表格
   - 包括项目ID和标题
   - 根据预算类别对支出进行分类
   - 获取必要的签名（团队负责人、指导教师）

3. **支持文档**
   - 1,000元以上购买的价格比较
   - 对任何不寻常或特殊购买的解释信
   - 如果申请注册费，则需要会议信息
   - 如果申请出版费，则需要已出版材料

### 步骤3：提交报销申请

1. 在财务平台上，导航至"报销申请"
2. 选择"学生项目报销"
3. 输入项目信息：
   - 项目ID
   - 项目标题
   - 项目类型（创新、创业训练等）
   - 指导教师姓名

4. 输入费用详情：
   - 费用类别
   - 金额
   - 购买日期
   - 供应商信息
   - 费用目的
   - 上传收据的扫描副本

5. 提交审批：
   - 指导教师将收到通知进行审批
   - 系部管理员将进行审查
   - 财务部门将处理已批准的申请

### 步骤4：跟进并收取报销款项

1. 在系统中检查报销申请的状态
2. 及时回应任何查询或额外信息请求
3. 一旦获批，资金通常会在7-10个工作日内转入指定账户

## 报销截止日期和限制

- **初始阶段（30%）**：必须在项目批准后3个月内申请
- **中期阶段（50%）**：必须在通过中期检查后3个月内申请
- **完成阶段（20%）**：必须在最终报告批准后1个月内申请
- **财政年度截止日期**：所有报销必须遵守大学的财政年度截止日期

## 预算管理技巧

1. **提前计划**：在项目开始时创建详细预算
2. **跟踪支出**：维持所有项目支出的持续记录
3. **保存收据**：安全存储所有收据并制作数字副本
4. **平衡支出**：在三个资金阶段分配支出
5. **记录变更**：保持对任何预算调整的书面说明
6. **大额购买前咨询**：在进行大额支出前咨询指导教师
7. **批量报销**：可能时一起提交多个支出
8. **保留应急资金**：为意外支出保留一些资金

## 常见问题及解决方案

### 收据丢失
- 对于小金额（200元以下），填写收据遗失表格
- 对于较大金额，尝试从供应商处获取副本
- 如果无法替换，提供详细解释和支持证据

### 预算类别调整
- 经过合理解释，允许类别之间的小调整
- 重大变更需要指导教师事先批准
- 在项目文档中记录所有变更

### 逾期提交
- 避免逾期提交，因为可能被拒绝
- 如果不可避免，提供书面解释说明延迟原因
- 延期请求必须在原始截止日期前提交

## 特殊考虑事项

### 团队购买
当团队成员使用个人资金进行购买时：
- 确保收据上有大学名称
- 记录哪个团队成员进行了购买
- 包括团队成员的银行账户信息以便报销

### 会议差旅
对于会议出席：
- 获得会议参与的预先批准
- 保存会议议程和注册确认
- 保留所有旅行收据（交通、住宿）
- 准备关于会议参与的简短报告

通过理解并遵循这些经费和报销指南，你可以确保你的项目在财务上保持健全，并且所有团队成员在整个项目生命周期中得到适当支持。
